# search-orm

Loopback like ORM built to work with any node.js framework.

* [Filters](#filters)
* [Relations](#relations)
* [Connectors](#connectors)

## Filters

### Base Filters

```javascript
{
  "where": {},
  "limit": 100,
  "skip": 2, // offset
  "search_after": [], // optional for pagination
  "order": [], // [] or ""
  "fields": [], // [] or ""
  "include": [], // [] or "" for relations
}
```
